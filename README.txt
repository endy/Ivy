################################################################################
    Ivy Graphics Prototyping Library
################################################################################

Ivy is a feature-lite (at the moment) graphics library undergoing slow but deliberate design towards a simple abstraction of the OpenGL and Direct3D graphics APIs.

Ivy grew out of a basic D3D11 application into a repository of code I've written for projects and experiments.  I plan to formalize this but until then feel free to peruse any of the code for your own use as guided by the licensing terms (see LICENSE.txt for details).
